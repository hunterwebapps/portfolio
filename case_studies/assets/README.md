# Case Study Assets

Drop screenshots, exported diagrams, dashboard snapshots, and short demo clips here. Reference them from the relevant case study markdown file with a relative link, for example:

```markdown
![Operator Console live stream](assets/01-operator-console.png)
```

## Suggested assets per case study

### 01 – AI Vision Agent
- Screenshot of the Operator Console showing the live desktop stream + synchronized action log
- Short clip of the virtual operator completing an EDI import end-to-end
- "Before/after" of a manual EDI import vs. the automated one (timing)

### 02 – Analytics + PIFOT
- Screenshot of the embedded Power BI report (PIFOT dashboard)
- Drill-down view by workcenter, customer, or channel
- A redacted retailer scorecard exported from the dashboard

### 03 – Ship List + ADaM Modernization
- Screenshot of the live Ship List grid with validation badges visible
- Side-by-side: legacy ERP screen vs. the Ship List
- A "near-miss" exception caught by validation before it became a chargeback

### 04 – Browser Extension
- ShipIQ page with PO highlighting active
- Amazon Vendor Central page with the extension active
- The Christy side panel docked next to a customer portal

### 05 – Automated Amazon Routing
- Timeline diagram of the end-to-end flow (email → confirmed shipment)
- A redacted Amazon transaction confirmation
- A "lights-out" timestamp showing the automation running outside business hours

Keep filenames short, lowercase, and prefixed with the case study number, e.g. `02-pifot-overview.png`.
